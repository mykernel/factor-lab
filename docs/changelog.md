# 版本记录

这里记录 Factor Lab 最近的公开功能变化。更完整的细节以线上平台“关于平台 / 版本记录”为准。

## v1.9 - 报告广场上线

发布时间：2026-06-08

- 研究助手新增“报告广场”入口。
- 报告可按股票归档展示，支持搜索、类型筛选和分页。
- 单股普通/快速深度报告可免费查看。
- 圆桌报告支持一次性解锁，解锁后反复查看不再重复扣费。

## v1.8 - 股票日线与断层回补观察

发布时间：2026-06-08

- 股票详情新增日线图。
- 支持查看信号日前后的价格走势、断层缺口和回补状态。
- 日线数据来自平台已有历史行情缓存，不改写股票池结果，也不额外消耗积分。

## v1.7 - 关联因子与配置说明优化

发布时间：2026-06-07

- 新增“断层后量能”筛选因子。
- 断层后量能会自动关联“是否利润断层=是”。
- 筛选因子和评分因子配置页新增因子说明。
- 因子说明改为弹窗展示，配置页更稳定。

## v1.6 - 公开分享页与报告阅读体验升级

发布时间：2026-06-05

- 公开分享页新增阅读人次展示。
- 分享页支持目录和标题锚点。
- 分享页新增 Factor Lab 介绍和注册入口。
- 报告展示统一升级 Markdown 渲染能力。
- 圆桌报告队列展示排队位置和预计等待时间。
- 注册流程调整为邀请码可选，公开分享页改为免费注册引导。

## 说明

Factor Lab 仍处于试用和持续打磨阶段。部分功能、数据流程和积分规则可能会继续调整。
